# Authify — Enterprise Authentication System

Questo repository contiene la soluzione completa per un sistema di autenticazione enterprise composto da due componenti principali:

- `backend/` — API RESTful implementata con Spring Boot (Java 21), gestione JWT, mail, e persistenza MySQL.
- `frontend/` — Single Page Application (React + Vite) per l'interfaccia utente.

Per informazioni dettagliate su come sviluppare, testare e distribuire ciascuna parte vedi i README dedicati:

- Backend: `backend/README.md`
- Frontend: `frontend/README.md`

- Sviluppo frontend:
